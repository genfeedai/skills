# Onboarding

Quick onboarding guide focused on creating your first AI-generated content in under 10 minutes.

## Installation

```bash
npx skills add genfeedai/skills/onboarding
```

## Usage

Ask for help getting started:

```
How do I use Genfeed?
```

```
Help me create my first content
```

```
What is this app?
```

## What It Covers

### Quick Start (5 Minutes)

1. Open Studio
2. Create a simple workflow (Prompt → Image Generator → Output)
3. Run and see results

### Key Concepts

- **Nodes**: Building blocks (input, AI, processing, output)
- **Edges**: Connections between nodes
- **Executions**: Running workflows
- **Credits**: Usage-based billing

### Common Workflows

- Simple image generation
- Image to video
- Talking head videos
- Multi-image video montages

### Tips & Shortcuts

- Keyboard shortcuts
- Cost optimization
- Template usage

## First Workflow Example

```
[Prompt] → [Image Generator] → [Output]
```

1. Add **Prompt** node - type your image description
2. Add **Image Generator** node - generates the image
3. Add **Output** node - collects the result
4. Connect the nodes left to right
5. Click **Run**
